# 🎵 MoodTunes – AI-Powered Music Recommendation System

## 📖 Overview

MoodTunes is an AI-powered music recommendation system that suggests songs based on a user's mood. The project uses Reinforcement Learning (Q-Learning) to improve future recommendations by learning from user interactions such as liking or disliking songs.

---

## ✨ Features

- 🎭 Mood-based music recommendations
- 🧠 Reinforcement Learning using Q-Learning
- 🔍 Search songs by title or artist
- ⚡ Energy and positivity (valence) based recommendations
- ❤️ Add songs to favorites
- 📱 Responsive web interface
- 🎵 Six mood categories

---

## 🛠️ Tech Stack

- Python
- HTML5
- CSS3

---

## 📂 Dataset

The project contains a curated music dataset with six mood categories:

- Happy
- Sad
- Energetic
- Calm
- Romantic
- Angry

Each song contains:
- Song Name
- Artist
- Mood
- Energy Score
- Valence Score

---

## 🧠 How It Works

1. The user selects a mood.
2. The system recommends songs from the dataset.
3. Users can like or unlike recommendations.
4. Q-Learning updates song scores based on feedback.
5. Songs with higher Q-values are recommended first over time.
